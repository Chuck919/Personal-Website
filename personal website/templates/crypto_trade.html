Introduction and Thought Process

KEEP IN MIND THIS IS NOT FINANCIAL ADVICE.

People use crypto trading bots for various reasons, primarily to capitalize on the fast-paced and volatile nature of the cryptocurrency market. These automated software programs are designed to execute trades based on predefined strategies and algorithms, enabling users to benefit from market opportunities 24/7 without constant manual monitoring. Crypto trading bots can process vast amounts of data, analyze multiple indicators simultaneously, and execute trades at high speeds, which can be challenging for human traders. Additionally, bots can help eliminate emotional biases and impulsive decisions, thus promoting disciplined trading. By utilizing these tools, traders aim to enhance their chances of achieving profitable outcomes while optimizing their time and resources in the highly competitive and ever-changing cryptocurrency landscape.

There exists many different crypto trading platforms out there, however, I wanted to experiment with my own trading bots. Hence, Crypto Trade. Currently, this project only supports Binance.US (becaues Binance is one of the most popular exchanges for crypto bots), but in the future, I hope to add in more exchanges. This project is also made to be ran on pythonanywhere, so a lot of functionality (websockets, rabbitMQ, celery, threading, etc.) are not implemented into the project. All code can be found in my Github repository for this project.
